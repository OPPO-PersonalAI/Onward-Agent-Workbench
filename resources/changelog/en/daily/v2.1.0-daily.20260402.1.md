# Onward Daily Build v2.1.0-daily.20260402.1

Initial tagged release for Onward 2, a VS Code-style desktop terminal workspace built with Electron and React for multi-task development workflows.

## New Features
- Provide a VS Code-style desktop shell for working across multiple terminal tasks in one window.
- Support 1, 2, 4, and 6 terminal layouts so users can watch and control several sessions at once.
- Include a Prompt Notebook for saving, organizing, and sending reusable prompts to terminals.
- Bundle integrated developer tools such as Project Editor, Git Diff, Git History, the browser panel, and Settings.
- Ship the first tagged Daily Build with cross-platform packaging targets and the automated update pipeline enabled.
